# UnityPong
Unity3D Pong implementation

## Features
  * Singleplayer VS AI
  * Multiplayer
    * Via LAN/WAN (UDP Broadcast)
    * Hotseat
  * Epic AI vs AI battle
 
 ## Controls
 
 ### General
 
   * **Spacebar** - launch ball from paddle
   * **Esc** - exit to main menu
   
 ### Red Paddle (Player-server by default)
 
   * **W** - Up
   * **S** - Down
   
 ### Blue Paddle (Player-client by default)
 
   * **Arrow Up**
   * **Arrow Down**
